## Install


1.**Clone the repo**

```
$ git clone https://github.com/zhanming0601/react-native-app.git
$ cd react-native-app
```
2.**Install dependencies**

```
$ npm install
$ brew install watchman
```
3.**Start Packager**

```
$ npm start
```
4.**Running on iOS:**

```
$ react-native run-ios
```
5.**Running on Android:**

```
$ react-native run-android
```
